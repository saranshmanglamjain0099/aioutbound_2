🔧 Quick Setup Guide

One-Time Setup Steps
1

Run Supabase Schema

Open supabase_schema.sql → Supabase Dashboard → SQL Editor → Run it.

2

Configure LiveKit + Gemini

Go to ⚙️ Settings → fill LiveKit URL, API Key, Secret + Google API Key → Save both.

3

Configure Vobiz SIP

Fill SIP Domain, Username, Password, Outbound Number → Save → click ⚡ Create SIP Trunk.

4

Customise AI Prompt

Go to ✏️ AI Prompt → Edit the system prompt for your business → Save.

5

Create Agent Profile

Go to 🤖 Agents → Create a profile with your business name + preferred voice.

6

Test with Single Call

📞 Single Call → enter your own number → Dispatch. You should hear the AI within 5–10s. ✅
